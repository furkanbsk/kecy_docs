# website-kecy

KecyAI Robotik Wiki — Türkçe robotik ve ROS 2 dokümantasyon sitesi.

İçerik Notion'dan içe aktarılmıştır. Tekrar içe aktarmak için:

```bash
node scripts/import-notion.mjs
```

Geliştirme (monorepo kökünden):

```bash
yarn workspace website-kecy start
yarn workspace website-kecy build
yarn workspace website-kecy typecheck
```
